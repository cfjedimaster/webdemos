# <video-bgshadow>

This is a simple web component that uses the excellent [ColorThief](https://lokeshdhakar.com/projects/color-thief/) library to add an animated background to videos that match their dominent color. It's based on the example they created here: <https://lokeshdhakar.com/projects/color-thief/#v3-observe>

Right now it doesn't support any arguments, but I could see adding values for the size of the shadow. 

You can run the demo here: <https://cfjedimaster.github.io/webdemos/video-bgshadow/>
